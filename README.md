# STudent EVent Manager

Built for the "Φοιτητική Εβδομάδα" festival in the Aristotle University of Thessaloniki, but can be reused for other festivals too.

## Usage
### Ruby & Gems Update
```bash
# Update rvm
$ rvm get head

# Update ruby
$ rvm list known
$ rvm use ruby-2.5

# Update gems
$ gems update
```
